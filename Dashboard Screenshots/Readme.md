# 🏥 Healthcare Analytics Dashboard (Power BI)

## 📌 Project Overview

The **Healthcare Analytics Dashboard** is an interactive Power BI solution developed to analyze healthcare data and transform raw information into meaningful business insights. The dashboard enables healthcare stakeholders to monitor patient demographics, disease trends, hospital performance, financial metrics, and admission activities through interactive visualizations.

---

## 🎯 Project Objectives

- Analyze patient demographics and healthcare trends.
- Monitor hospital and doctor performance.
- Track patient admissions and discharges.
- Evaluate financial performance using billing data.
- Support data-driven healthcare decision-making.

## 💡 Executive Summary

This dashboard analyzes healthcare data collected between **2022 and 2024**, providing valuable insights into **patient demographics, disease trends, hospital performance, financial metrics, admissions, and discharges**. It enables healthcare organizations to monitor operational performance, identify key trends, and support **data-driven decision-making**. The analysis also highlights **COVID-19 as the most reported disease**, emphasizing its significant impact on overall patient admissions during the analysis period.
